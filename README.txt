=== Insta Post ==== 

Experimental plugin - USE AT OWN RISK.

====================================================================================

Requires Advanced Custom Field https://wordpress.org/plugins/advanced-custom-fields/

====================================================================================

1. Install and activate as usual.

2. Import the ACF JSON Group.

3. Add new Post

4. Add new Instagram post from repeater field

5. Paste Instagram post URL -- e.g. https://www.instagram.com/p/BMucx5NBRua/
